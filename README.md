🚀 **Features**

- ⚡ Super-fast conversions (integer-based)  
- 🤓 Friendly for students & beginners  
- 🐧 Cross-platform: Linux, Windows, Mac (with a C compiler)  
- 🎉 Pure C — no external libraries
